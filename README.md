# Paste App  
Live Demo: https://paste-app-seven-snowy.vercel.app

Paste App is a simple code-saving web application built with React and Vite.  
It allows users to store code snippets locally and manage them for future use.

---

## Features

- Save code snippets in the browser
- Persistent storage using Local Storage
- View saved code pastes
- Edit existing pastes
- Copy code to clipboard
- Delete unwanted pastes
- Search pastes by title
- Fast performance with Vite
- Clean UI using Material UI

---

## Tech Stack

- React
- Vite
- Redux Toolkit
- Material UI
- React Router
- Local Storage
- Vercel (Deployment)
